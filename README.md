**Blog Management - REST API**

A blog management created by [NestJs](https://nestjs.com) framework.

# Project setup

```bash
npm install
```

# Compile and run the project

```bash
# development mode:
npm run start

# watch mode:
npm run start:dev

# production mode:
npm run start:prod
```

# Run tests

```bash
# unit tests:
npm run test

# e2e tests:
npm run test:e2e

# test coverage:
npm run test:cov
```
